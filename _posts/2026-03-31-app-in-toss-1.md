앱인토스(Apps in Toss)를 처음 검토할 때 필요한 내용을 공식 문서와 커뮤니티 기준으로 정리했다.  
핵심 개념, 사업자 등록 필요 여부, 오픈 절차를 중심으로 확인했다.

---

## 앱인토스란?

앱인토스는 **토스 앱 내에서 외부 서비스가 별도 설치 없이 실행되는 앱인앱 구조의 미니 플랫폼**이다.

주요 특징은 다음과 같다.

- 즉시 접근: 앱 설치 과정 없이 토스 앱 내에서 바로 실행
- 통합 인증: 토스 계정 기반으로 별도 가입 절차를 단순화
- 결제 연동: 토스페이를 포함한 결제 흐름 구성 가능
- 개발 도구 제공: TDS(토스 디자인 시스템), TUBA(분석 도구), SDK 등 지원

---

## 사업자 관점에서의 장점

앱인토스가 사업자에게 매력적인 이유는 아래와 같다.

1. 현재 서비스 수수료가 없음
2. 누적 가입자 3,000만 명 사용자에게 앱 설치 없이 서비스 노출 가능
3. 설치 장벽이 낮아 초기 유입/전환 검증에 유리

사례로, 인디게임 개발사 마나바바(돌돌디)가 앱인토스 제휴 후 월 매출 2억 1천만원을 달성했다는 보도가 있다.

- 출처: [https://indiegame.com/archives/21653](https://indiegame.com/archives/21653)

## 사업자 등록증, 꼭 필요한가?

**사업자 등록 없이도 미니앱 런칭은 가능**하다.  
다만 계약/정산 및 법적 요건이 필요한 기능은 사업자 등록 후 이용할 수 있다.

### 사업자 없이 가능한 기능

- 세그먼트(유저 그룹핑)
- 푸시/알림
- 게임 프로필, 리더보드
- 공유 리워드

### 사업자 등록 후 가능한 기능

- 토스 로그인
- 비즈월렛
- 프로모션
- 인앱 광고
- 토스 페이
- 인앱 결제

정리하면, 초기 서비스 검증 단계는 사업자 없이 진행 가능하고, 수익화/정산 연동 단계에서 사업자 등록이 필요하다.

- 출처: [https://techchat-apps-in-toss.toss.im/t/topic/1666](https://techchat-apps-in-toss.toss.im/t/topic/1666)

## 서비스 오픈 프로세스

개발자 센터 문서 기준으로 서비스 오픈 절차는 다음 4단계다.

1. **콘솔에서 앱 등록하기**
   - 토스 비즈니스 가입: [https://business.toss.im/](https://business.toss.im/)
   - 앱 등록 진행
   - 게임의 경우 배포 방식은 WebView / React Native / Unity 지원

   ![앱인토스 콘솔 앱 등록 화면](/assets/img/for_post/2026-03-31-app-in-toss-1.png)
   ![앱인토스 가이드 추천 화면](/assets/img/for_post/2026-03-31-app-in-toss-2.png)

2. **디자인하기**
   - 앱빌더(웹 기반) 또는 피그마
3. **개발하기**
   - SDK 2.x 사용
4. **검수하기**
   - 운영 -> 기능 -> 디자인 -> 보안 순
   - 평균 2~3일 소요
   - 게임은 추가로 등급 심의 필수
     - 게임물관리위원회 심의 또는
     - 앱스토어/플레이스토어 선출시 후 링크 첨부

- 출처: [https://developers-apps-in-toss.toss.im/intro/onboarding-process.html](https://developers-apps-in-toss.toss.im/intro/onboarding-process.html)
